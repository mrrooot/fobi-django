It's easy to override a theme!

You have to do as follows in order to do so:

  1. Create an app for holding the overridden theme.
  2. Inherit the theme you want to override.
  3. Register it in the registry with `force` argument set to True.
  4. Override some of the templates. Take this one as a good example.

Read the documentation for more information.
